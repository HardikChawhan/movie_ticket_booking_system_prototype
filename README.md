
# Movie ticket booking system

This is a simple movie ticket booking system written in C. It allows users to book seats for different showtimes, view the seating chart, and calculate the total price.

- [Getting Started](#getting-started)
- [Usage](#usage)
- [File Structure](#file-structure)

## Getting Started

### Prerequisites

- C compiler (e.g., GCC)

### Installation

1. Clone the repository to your local machine:

   ```sh
   git clone https://github.com/HardikChawhan/movie_ticket_booking_system_prototype.git
   cd movie-ticket-booking

2. Compile the program: 
    ```sh
    gcc movie_booking_system.c -o movie_booking_system.c

3. Run the program:
    ```sh
    ./movie_booking_system

## Usage

- Choose from different showtimes and interact with the program through the menu.
- You can book seats, display the seating chart, and calculate the total price.
- The seating chart data is saved and loaded from separate .dat files for each showtime.

## File Structure


- movie_booking_system.c : The main C code for the movie ticket booking system.
- seating_chart_*.dat: Data files storing the seating chart for different showtimes.
- README.md: This README file.
- flowchart.png: A flowchart diagram illustrating the programs logic.
